# 🎓 AI Voice Learning Tutor

An AI-powered Multi-Agent Learning Assistant built using **React, FastAPI, Python, and OpenRouter API**. The application helps students learn through intelligent teaching, quiz generation, study notes, voice interaction, and PDF export.

---

## 🚀 Live Demo

### 🌐 Frontend (Vercel)
https://ai-voice-learning-tutor.vercel.app/

### ⚙️ Backend (Render)
https://ai-voice-learning-tutor.onrender.com/

---

## ✨ Features

- 🧠 AI Teaching Agent
- 📚 Resource Recommendation Agent
- ❓ Quiz Generation Agent
- 📝 Smart Notes Generator
- 📄 PDF Notes Download
- 🎤 Voice Input (Speech Recognition)
- 🔊 AI Voice Output (Text-to-Speech)
- 🌙 Modern Responsive Dark UI
- ⚡ FastAPI REST API Backend
- ☁️ Deployed on Render & Vercel

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Ashmitha2802/AI-Voice-Learning-Tutor.git
```

```bash
cd AI-Voice-Learning-Tutor
```

---

## Backend Setup

```bash
cd backend
```

Create a virtual environment

```bash
python -m venv venv
```

Activate

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
uvicorn main:app --reload
```

---

## Frontend Setup

```bash
cd frontend
```

Install packages

```bash
npm install
```

Run

```bash
npm run dev
```
